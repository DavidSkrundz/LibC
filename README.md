====

A limited Swift wrapper around CLibC


Features
--------

- Random Integers
- System Date/Time Info
- Get/Set Working Directory
- List Hardware Interfaces
- Open Sockets


Requires On Linux
-----------------
`$ sudo apt-get install libbsd-dev`
